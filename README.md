# Ai私域 控制台

这是一个可直接发布到 GitHub Pages 的静态网页原型。

## 文件说明

- `index.html`: 主页面

## 发布到 GitHub Pages

1. 在 GitHub 新建一个仓库，例如 `ai-private-domain-console`
2. 把当前目录里的 `index.html` 上传到仓库根目录
3. 进入 GitHub 仓库页面
4. 打开 `Settings`
5. 打开 `Pages`
6. 在 `Build and deployment` 中选择：
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/ (root)`
7. 保存后等待几十秒到几分钟
8. GitHub 会生成一个公开链接，格式通常类似：

```text
https://你的用户名.github.io/ai-private-domain-console/
```

## 本地预览

直接双击 `index.html`，或在浏览器中打开即可。
